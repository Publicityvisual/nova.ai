## ⚙️ CONFIGURACIÓN RÁPIDA

### 1. Variables de Entorno (.env)

```bash
# Telegram
TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_BOT_TOKEN_2=backup_1
TELEGRAM_BOT_TOKEN_3=backup_2

# AI (OpenRouter principal)
OPENROUTER_API_KEY=your_key
GROQ_API_KEY=optional
FIREWORKS_API_KEY=optional

# Pagos
STRIPE_SECRET_KEY=sk_live_...
MERCADOPAGO_ACCESS_TOKEN=...

# Config
OWNER_NUMBER=your_number
BOT_NAME=Sofia
NODE_ENV=production

# Database
DATABASE_URL=sqlite://./data/nova.db

# Monitoring
HEALTH_CHECK_SECRET=random_secret
```

### 2. Configurar GitHub Secrets

```bash
Ir a: Settings > Secrets > Actions

Agregar:
- RENDER_DEPLOY_HOOK (from Render dashboard)
- RENDER_SERVICE_URL (https://nova-ai-pro.onrender.com)
```
