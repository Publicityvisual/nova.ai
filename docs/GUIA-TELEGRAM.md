# 🤖 SOFIA EN TELEGRAM

## 🚀 CONFIGURACIÓN RÁPIDA

### Paso 1: Crear Bot en Telegram
1. Abre Telegram y busca **@BotFather**
2. Envía: `/newbot`
3. Elige nombre: `Sofia Gonzalez` (o tu preferencia)
4. Elige username: `tuempresa_bot` (debe terminar en _bot)
5. **Copia el TOKEN** que te da

### Paso 2: Configurar Token
Edita el archivo `.env` y agrega:
```
TELEGRAM_BOT_TOKEN=tu_token_aqui
TELEGRAM_ADMIN_CODE=sofia-admin-2025
```

### Paso 3: Reiniciar Sofia
```bash
REINICIAR-SEGURO.bat
```

Verás: `✅ TELEGRAM BOT: Conectado y listo`

### Paso 4: Autorizarte como Admin
En Telegram, escribe:
```
/auth sofia-admin-2025
```

Listo ✅ Ahora tienes control total.
