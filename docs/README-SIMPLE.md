# 🚀 NOVA AI - SIMPLIFICADO

## 📱 SOLO 3 ARCHIVOS NECESARIOS:

```
Nova AI/
├── 🚀 INICIAR.bat          ← UNICO archivo para iniciar
├── 📋 CONFIGURAR.txt       ← Guía de configuración
└── README-SIMPLE.md        ← Esta guía
```

## ⚡ PASOS RAPIDOS:

### 1. CONFIGURAR (Solo una vez):
- Edita archivo `.env` con tus datos
- Token de Telegram: TELEGRAM_BOT_TOKEN=tu_token

### 2. INICIAR (Siempre):
- Doble click en: `🚀 INICIAR.bat`
- Espera 10-20 segundos
- Listo ✅
