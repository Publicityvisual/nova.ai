# 🧠 SOFIA INTELLIGENCE v4.0
## Sistema de Adaptación Psicológica Dinámica

Sofia ahora utiliza un sistema de **inteligencia artificial de élite** que analiza el perfil psicológico de cada persona y adapta su respuesta **automáticamente**.

---

## 🎯 ¿Qué hace diferente a Sofia?

### Antes (Sistema viejo):
- Sofia tenía una personalidad única y rígida
- Respondía igual a todos
- A veces no conectaba con ciertos tipos de clientes

### Ahora (Sistema Inteligente v4.0):
- **Detecta automáticamente** qué tipo de persona le está escribiendo
- **Se transforma** instantáneamente para conectar con ellos
- Usa **4 dimensiones psicológicas** para entender al cliente

## 🧬 Las 4 Dimensiones Psicológicas

### 1. **Formalidad** (0-100%)
- **0-40%** → Usa "tú", jerga mexicana natural, relajada
- **40-70%** → Balance profesional-cercano
- **70-100%** → "Usted", vocabulario corporativo elegante

### 2. **Nivel Técnico** (0-100%)
- **0-40%** → Explica conceptos técnicos simple
- **40-70%** → Términos técnicos con contexto
- **70-100%** → Jerga técnica avanzada sin explicar (asume que sabe)

### 3. **Nivel Emocional** (0-100%)
- **0-40%** → Lógica, datos, estructura
- **40-70%** → Balance racional-emocional
- **70-100%** → Storytelling, metáforas, pasión, conexión emocional

### 4. **Directitud** (0-100%)
- **0-40%** → Exploratoria, pregunta opciones, detalla
- **40-70%** → Balance explicación-directitud
- **70-100%** → Directa, corta, accionable, va al grano

## 👥 Perfiles Detectados Automáticamente

### 1. **Executive Técnico** 🖥️
*"El ingeniero/director que sabe de tecnología"*

**Detectado por:**
- Usa "usted" o "tú" formal
- Menciona términos técnicos (API, SEO, React, etc.)
- Estructura de mensajes organizada
- Pregunta específicas técnicas

**Cómo responde Sofia:**
```
"Entiendo que necesitas cotización. Para precisión, necesito entender 
el alcance técnico:

1. ¿Requieres solo frontend o full-stack?
2. ¿Tienes especificaciones técnicas definidas?
3. ¿Hay integraciones con APIs de terceros?

Basándome en proyectos similares, el rango es $25k-$80k MXN. 
¿Te funciona una llamada técnica de 20 min el jueves?"
```

**Características:**
- ✅ Usa bullets y numeración
- ✅ Menciona tecnologías específicas
- ✅ Da datos concretos (rangos, tecnologías)
- ✅ Propone agendas técnicas
- ❌ No usa emojis (o máximo 1 técnico)
- ❌ No explica conceptos básicos

## ⚙️ Cómo Funciona el Sistema

### 1. **Detección en Tiempo Real**
```javascript
// Cuando llega un mensaje:
analyzeUserProfile(userId, message, history)
  ↳ Calcula: formalidad, técnico, emocional, directitud
  ↳ Detecta: industria, urgencia
  ↳ Clasifica: 1 de 8 perfiles
```

### 2. **Adaptación Dinámica**
```javascript
// Genera system prompt personalizado:
generateAdaptedSystemPrompt(profile)
  ↳ Cambia: tono, vocabulario, estructura
  ↳ Ajusta: uso de emojis, saludos, despedidas
  ↳ Selecciona: ejemplos del mismo estilo
```

### 3. **Memoria del Perfil**
```javascript
// Guarda perfil del usuario:
userProfiles.set(userId, {
  profile: { formalidad: 75%, técnico: 60%... },
  personalityType: 'executive_technical',
  messages: [...],
  preferences: {...}
})
```

### 4. **Respuesta Inteligente**
```javascript
// Genera respuesta según perfil:
getIntelligentResponse(profile, intent)
  ↳ executive_technical → bullets + términos técnicos
  ↳ emotional_creator → storytelling + pasión
  ↳ direct_buyer → precios claros sin relleno
```
