## 💬 EJEMPLOS DE USO REAL

### Ejemplo 1: Investigar y Enviar Email Automático

**Usuario:** *"Sofía, investiga sobre tendencias de branding 2025 y mándame un email con un resumen"*

**Sofia hace:**
1. ✅ Usa `deep_research` sobre "tendencias branding 2025"
2. ✅ Busca en múltiples fuentes con `web_search`
3. ✅ Extrae contenido con `web_fetch`
4. ✅ Genera resumen ejecutivo
5. ✅ Crea email profesional con `send_email`
6. ✅ Lo envía al usuario

**Resultado:** Email completo en bandeja de entrada en 2-3 minutos

---

### Ejemplo 2: Procesar Audio y Responder

**Usuario envía audio:** *"Hola Sofía, necesito urgente una cotización para un logo y sitio web para mi restaurante que abre el mes que viene"*

2. ✅ Detecta: "urgente", "restaurante", "logo", "sitio web", "mes que viene"
3. ✅ Clasifica: Executive Relacional + Urgent Client
4. ✅ Busca en base: patrones de éxito para restaurantes
5. ✅ Genera respuesta personalizada:

```
"¡Qué emoción que vayas a abrir tu restaurante! 🍽️✨ 
Entiendo la urgencia, así que voy directo:

Para restaurantes, recomiendo:
• Logo completo con manual de marca: $12k (incluye versión para menú, uniformes, redes)
• Sitio web con reservas: $25k (optimizado para móvil, Google Maps, galería de platillos)
• Pack redes (primer mes): +$8k

Para abrir en un mes necesitamos empezar YA. 
¿Te late una llamada rápida hoy para definir el mood de tu marca? 🎨

"
```

## 🛠️ SISTEMA DE PLUGINS

### Cómo crear un plugin:

```javascript
// archivo: mi-plugin.js
module.exports = {
  name: 'mi_herramienta',
  description: 'Hace algo increíble',
  parameters: {
    param1: 'string',
    param2: 'number'
  },
  execute: async (params) => {
    // Tu lógica aquí
    return { 
      success: true, 
      result: 'listo!' 
    };
  }
};
```

### Cargar plugin:
```javascript
universalConnector.loadPlugin('./mi-plugin.js');
```

### Usar plugin:
```javascript
universalConnector.executeTool('mi_herramienta', {param1: 'valor'});
```
