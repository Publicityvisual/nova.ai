### 💬 TELEGRAM (Para control completo)

#### 🧰 Control del Sistema (Admin)
```
/sistema           → 💻 Info completa del PC
/captura           → 📸 Screenshot en segundos
/procesos          → 🔍 Ver procesos activos
/reiniciar         → 🔄 Reiniciar Sofia
/sesiones          → 📱 Estado de conexiones
/version           → ℹ️  Versión instalada
```

#### 🔍 Información y Búsquedas
```
/buscar [texto]                      → 🔍 Buscar en internet
Ejemplo: /buscar tendencias diseño 2025

/clima [ciudad]                      → 🌤️  Ver clima
Ejemplo: /clima Ciudad de México

/moneda [cantidad] [de] [a]           → 💱 Conversor
Ejemplo: /moneda 100 USD MXN

/estado                              → 📊 Estado general
```

#### 📧 Comunicación y Productividad
```
/email [para]|[asunto]|[mensaje]      → 📧 Enviar email
Ejemplo:
/email cliente@ejemplo.com|Título|Mensaje aquí...

/investigar [tema]                   → 📚 Investigar y generar PDF
Ejemplo: /investigar marketing digital 2025

/recordatorio [tiempo] [mensaje]     → ⏰ Crear recordatorio
Ejemplo: /recordatorio 30m Llamar al cliente
```

#### 🛡️ Mantenimiento (Solo Admin)
```
/backup             → 💾 Crear backup completo
/limpiar            → 🧹 Limpiar caché y logs
/diagnostico        → 🔍 Diagnóstico completo
```
