# 🤖 SOFIA EN WHATSAPP - COMANDOS COMPLETOS

## ✅ WhatsApp AHORA TIENE todas las herramientas OpenClaw

### 📱 Comandos Disponibles en WhatsApp

#### 🔧 Control del Sistema
```
/admin captura                    → 📸 Screenshot del PC
/admin sistema                    → 💻 Info básica del sistema
/admin sistema-completo           → 💻 Info DETALLADA del sistema
/admin status                     → 📱 Estado WhatsApp
/admin sesiones                   → 💾 Ver sesiones guardadas
/admin reporte                    → 📊 Enviar reporte diario
/admin reiniciar                  → 🔄 Reiniciar Sofia
/admin backup                     → 💾 Crear backup completo
/admin limpiar                    → 🧹 Limpiar caché y optimizar
```

#### 🌐 Herramientas Web
```
/admin buscar [texto]             → 🔍 Buscar en internet
Ejemplo: /admin buscar tendencias diseño 2025

/admin clima [ciudad]              → 🌤️ Ver clima
Ejemplo: /admin clima Querétaro
Ejemplo: /admin clima Ciudad de México

/admin moneda [cantidad] [de] [a]  → 💱 Conversor de divisas
Ejemplo: /admin moneda 100 USD MXN
Ejemplo: /admin moneda 50 EUR USD
```

#### 📧 Comunicación
```
/admin email [para]|[asunto]|[mensaje]  → 📧 Enviar email
Ejemplo:
/admin email cliente@email.com|Cotización|Hola, te envío la cotización solicitada para el proyecto de branding. Quedo atenta a tus comentarios.
```

## 🚀 COMANDOS COMBINADOS (Usa ambos)

### Flujo de trabajo típico:

1. **Cliente te escribe en WhatsApp:**
   - "Necesito una cotización"

2. **Investigas en el momento:**
   - Tú: `/admin buscar precios diseño web 2025`
   - Sofia te da información al instante

3. **Revisas el clima para la reunión:**
   - Tú: `/admin clima Querétaro`
   
4. **Capturas tu pantalla del trabajo:**
   - Tú: `/admin captura`
   - Envías al cliente: "Así va quedando..."

5. **Envías email formal desde WhatsApp:**
   - Tú: `/admin email cliente@email.com|Cotización|Adjunto la propuesta...`

**¡Todo sin abrir la computadora!** 📱
