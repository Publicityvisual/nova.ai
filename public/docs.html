Documentación API NovaAI

Bienvenido a la documentación de la API de NovaAI. Nuestra API es compatible con OpenAI, lo que significa que puedes usarla con el mismo código que ya tienes, solo cambiando la URL base.

Base URL: https://novaai-38a4e.web.app

Autenticación

Todas las llamadas a la API requieren autenticación mediante Bearer token en el header Authorization.

Header de Autenticación 
Authorization: Bearer TU_API_KEY

Obtén tu API key desde el Dashboard.

Chat Completions

POST /v1/chat/completions

Crea una conversación con cualquiera de nuestros modelos de lenguaje.

Request Body

Parámetro Tipo Requerido Descripción
model string REQUERIDO ID del modelo (e.g., "nova-1")
messages array REQUERIDO Lista de mensajes de la conversación
temperature number Opcional 0-2, default: 0.7
max_tokens integer Opcional Máximo de tokens a generar

Ejemplo de Request

curl 
curl https://novaai-38a4e.web.app/v1/chat/completions \
  -H "Authorization: Bearer TU_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nova-1",
    "messages": [
      {"role": "system", "content": "Eres un asistente útil."},
      {"role": "user", "content": "¿Qué es NovaAI?"}
    ],
    "temperature": 0.8,
    "max_tokens": 500
  }'

Image Generation

POST /v1/images/generations

Genera imágenes desde descripciones de texto.

Request Body

Parámetro Tipo Requerido Descripción
prompt string REQUERIDO Descripción de la imagen
n integer Opcional Número de imágenes (1-4)
size string Opcional Tamaño: 1024x1024, 512x512
curl 
curl https://novaai-38a4e.web.app/v1/images/generations \
  -H "Authorization: Bearer TU_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Un dragón majestuoso volando sobre montañas nevadas",
    "n": 1,
    "size": "1024x1024"
  }'

Códigos de Error

Código Descripción
400 Bad Request - Parámetros inválidos
401 Unauthorized - API key inválida o ausente
429 Rate Limit - Créditos insuficientes o límite alcanzado
500 Internal Error - Error del servidor

Límites y Rate Limits

  • Plan Free: 100 requests/mes, $10 créditos
  • Plan Pro: Requests ilimitadas, $50 créditos/mes
  • Timeout: 60 segundos por request
  • Max tokens: 4096 por request

Ejemplos de Uso

JavaScript/Node.js

JavaScript 
const response = await fetch('https://novaai-38a4e.web.app/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer TU_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'nova-1',
    messages: [{ role: 'user', content: 'Hola' }]
  })
});

const data = await response.json();
console.log(data.choices[0].message.content);

Python

Python 
import requests

response = requests.post(
    "https://novaai-38a4e.web.app/v1/chat/completions",
    headers={
        "Authorization": "Bearer TU_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "nova-1",
        "messages": [{"role": "user", "content": "Hola"}]
    }
)

data = response.json()
print(data["choices"][0]["message"]["content"])
